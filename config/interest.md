# PaperPuller Interest Profile

I am interested in papers about computer vision and visual recognition, especially:

1. OCR, document understanding, text detection, and text spotting.
2. Scene text recognition and robust recognition under difficult visual conditions.
3. Vision Transformers, transformer-based visual recognition, and efficient ViT variants.
4. Masked Autoencoders, self-supervised visual pretraining, and representation learning.
5. Data augmentation, synthetic data generation, domain randomization, and robustness for recognition systems.

Prefer papers with clear method, dataset, benchmark, or deployment value.

Down-rank papers that are only about robotics, pure LLM prompting, medical imaging, 3D reconstruction, or general machine learning without a meaningful visual recognition connection.
